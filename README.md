# 🧠 Mansparsh – Mental Health Management System

## 📌 Overview

**Mansparsh** is a digital mental health management platform designed to support individuals in monitoring, managing, and improving their emotional well-being. The system provides a structured environment where users can track moods, access resources, and seek support when needed.

---

## 🎯 Objectives

* Promote mental health awareness
* Provide a safe space for users to express emotions
* Enable self-monitoring through mood tracking
* Offer accessible mental health resources

---

## 🚀 Features

### 👤 User Module

* User Registration & Login (Secure Authentication)
* Profile Management
* Daily Mood Tracking
* Personal Notes / Journaling

### 📊 Dashboard

* Visual representation of mood trends
* Activity insights

### 💬 Support System

* Access to mental health resources
* Guidance and self-help content

### 🔒 Security

* Password encryption
* Secure session handling

---

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/your-username/mansparsh.git

2. Open project in Apache NetBeans

3. Configure Database:

* Create MySQL database
* Import SQL file

4. Update database credentials

5. Run using Apache Tomcat
